# SteflixPictures

**🔗 Live Demo:** [View the website here](https://catinca2005.github.io/Steflix-Pictures/)

---

**SteflixPictures** is a digital platform for cinematic adaptations of Romanian literature, developed as a high school certificate project.

## Purpose

* **Engagement:** Encourages student involvement through interactive media.
* **Education:** Facilitates access to creative materials for Romanian literature.
* **Skills:** Promotes linguistic, literary, and digital competencies.

## HTML & Tech Overview

The project is designed to be lightweight and user-friendly:

* **Framework:** Built with **HTML5** and **W3.CSS** to ensure the site is fully responsive on both mobile and desktop.
* **Optimization:** Uses **YouTube embeds** for video playback, ensuring fast loading speeds without hitting storage limits.
* **Interactivity:** Includes custom **CSS animations** for image galleries and smooth scrolling for navigation.
